# CFPV Knowledge Base

## Project

CFPV is a cross-platform Food & Beverage mobile application.

Platforms:

* Android
* iOS

## Tech Stack

Frontend:

* Flutter

Backend:

* NestJS

Database:

* PostgreSQL

Cache:

* Redis

## Design System

DESIGN.md is the single source of truth.

Requirements:

* Warm Starbucks-inspired design
* Green Accent (#00754A) for primary CTA
* House Green (#1E3932) for dark surfaces
* Neutral Warm (#f2f0eb) page background
* 50px pill buttons

## Development Workflow

Always follow:

1. Spec first
2. Plan second
3. Implement third
4. Test fourth
5. Review before merge

## Quality Standards

* Null-safe code
* Unit tests required
* Widget tests required
* Responsive layouts required
* Accessibility support required
